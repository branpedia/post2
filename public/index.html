API Integration

Gunakan endpoint berikut untuk mengambil screenshot secara terprogram:

GET /api/screenshot?url={URL}&device={device}

Parameter:

  • url: URL website yang akan di-screenshot (wajib)
  • device: Jenis perangkat (desktop, tablet, phone) - opsional, default: desktop

Contoh:

https://your-vercel-app.vercel.app/api/screenshot?url=https://github.com&device=phone
